# Security Policy

## Supported version

Only the newest Public Testnet Release Candidate is actively supported.

## Reporting a vulnerability

Do not open a public issue containing:

- private keys or wallet backups
- exploitable bridge or wallet vulnerabilities
- remote-code-execution details
- personal information
- live server credentials

Use a private GitHub Security Advisory when enabled for the repository. Provide a minimal reproduction, affected version and expected impact.

## Scope warning

Logicoin is experimental testnet software and has not received an independent professional audit. Do not use it to custody significant real-world value.
